Synthetic Grid API
===================

Deterministic synthetic power grid simulation framework.

This documentation provides a complete API reference for the system, including:

- Grid specification (`GridSpec`)
- Time series generation (`TimeSeries`, `TimeSeriesConstructor`)
- Configuration attributes (`TimeSeriesAttributes`)
- YAML config loader (`YamlConfigLoader`)
